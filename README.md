## Progetto Android — Turista Felice

**Corso:** Programmazione di Dispositivi Mobili (A.A. 2019/2020)

Applicazione Android che supporta l'esplorazione turistica di una città suggerendo luoghi di interesse nelle vicinanze. Funzionalità principali:

- Rilevazione della posizione tramite GPS del dispositivo
- Notifiche automatiche quando ci si avvicina a un punto di interesse
- Integrazione con **Google Places API** per la ricerca dei POI
- Integrazione con **Wikipedia API** per recuperare informazioni descrittive
- Filtraggio dei luoghi in base agli interessi dell'utente
- Salvataggio dei luoghi preferiti per pianificazione anticipata

**Team Gaucho:** Tommaso Sancini, Oleg Stoianov, Lorenzo Nosotti, Matteo Pastori, Andrea Moscatelli

## Prolog & Lisp — LMC Simulator

**Corso:** Linguaggi di Programmazione

Implementazione di un simulatore del **Little Man Computer (LMC)** — una macchina assembly semplificata a scopo didattico — realizzata in due linguaggi di programmazione dichiarativi:

- **`Prolog/lmc.pl`** — interprete LMC scritto in Prolog. Legge un file assembly, costruisce la memoria iniziale risolvendo etichette e istruzioni, ed esegue il programma ciclo per ciclo tramite `lmc_run/3`.
- **`Lisp/lmc.LISP`** — interprete LMC scritto in Common Lisp. Parsing del file assembly, costruzione della memoria e loop di esecuzione tramite `lmc-run`.

**Gruppo:** Andrea Moscatelli (833139), Matteo Pastori (829863)
